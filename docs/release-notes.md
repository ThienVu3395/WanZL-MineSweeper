# Release Notes - v1.0.0

## Features
- Full MineSweeper gameplay implementation
- Multiple difficulty levels
- Custom difficulty support
- Timer and best time tracking
- Persistent user preferences

## UI/UX
- Improved board interactions (hover, pressed states)
- Custom endgame dialog
- Keyboard shortcuts
- Visual indicators for exploded mines and incorrect flags

## Architecture
- MVVM pattern with RelayCommand
- Separation of domain models and UI logic
- Dedicated services for persistence and dialogs

## Testing
- Unit tests for core models, view models, and helpers
- High coverage for critical logic paths

## Documentation
- Full bilingual (EN/VI) XML documentation
- Structured development log
