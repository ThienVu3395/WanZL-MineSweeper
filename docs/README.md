# MineSweeper

A desktop MineSweeper game built with .NET and WPF.

## Project Overview

This project is being developed as a product-oriented technical assessment.
The goal is not only to create a functional MineSweeper game, but to build a clean,
maintainable, and user-friendly desktop application.

## Solution Structure

- `MineSweeper.App`  
  WPF desktop application responsible for the user interface.

- `MineSweeper.Core`  
  Core domain models and game logic.

- `MineSweeper.Tests`  
  Automated tests for core behaviors and game rules.

- `docs/`  
  Project documentation, architecture notes, development progress, etc.

## Current Status

Initial project structure and core models have been created.
Gameplay logic and UI implementation will be added incrementally in small commits.

## Technology Stack

- .NET
- WPF
- xUnit
